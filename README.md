# RAU: The Compressor

RAU is a file compression tool that uses the **Huffman Algorithm** to compress any text file.

This project was created to meet the requirements of the **Data Structures and Algorithm Classes** at **NED University of Engineering and Technology**. Building this tool is a great way to learn and understand how real-world compression works.
